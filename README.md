# 🌴 Caribee App - Your Ultimate Caribbean Travel Companion

**Caribee App** is a high-end, comprehensive travel platform designed to help users explore, experience, and navigate the Caribbean with ease. Built with the **Flutter** framework and **GetX** state management, it provides a seamless, high-performance experience for modern travelers.

---

## ✨ Key Features

* **🌅 Interactive Onboarding**: A multi-step, visually engaging introduction to help users understand the app's core value propositions.
* **🔐 Secure Authentication**: Full security suite including user registration, login, OTP (One-Time Password) verification, and robust password reset flows.
* **📍 Caribbean Exploration**: Dedicated modules to discover local events, top-rated restaurants, and unique island experiences.
* **📱 Integrated eSim Functionality**: Travel connectivity made easy with in-app eSim support.
* **🗺️ Advanced Map Integration**: Interactive map views to help users visualize and navigate to their favorite Caribbean destinations.
* **🛡️ Safety & Support**: Features like "Safe with Caribee" and dedicated help/support sections to ensure a worry-free trip.
* **👤 Comprehensive Profile Management**: Users can manage their personal data, update security settings, and view favorite saved locations.
* **🌓 Theme & Localization**: Supports flexible UI handling and localized strings for a personalized experience.

---

## 🛠️ Technology Stack

* **Frontend**: Flutter (Dart)
* **State Management**: GetX (Controller-based architecture)
* **Navigation**: GetX Routing for smooth screen transitions
* **Responsiveness**: `flutter_screenutil` for a consistent UI across all device sizes.
* **Utilities**: Shared Preferences (Local Storage), Dio/HTTP (Networking), and Lottie Animations.

## 📁 Project Structure

```text
lib/
├── controllers/      # Network listeners and global logic
├── core/             # Application routing and dependency injection
├── utils/            # Theme constants, colors, icons, and asset strings
└── view/             # The UI Layer
    ├── components/   # Globally reusable custom widgets
    └── screens/      # Feature-specific modules (Auth, Home, eSim, Profile)
```
